# HydroSense VES

Schlumberger Vertical Electrical Sounding (VES) Analysis Tool for Groundwater Exploration.

## Status
🚧 Under Development

## What it does
- Computes apparent resistivity (ρa) from field measurements
- Classifies VES curve types (A, H, K, Q)
- Visualizes sounding curves on log-log plots
- Exports results as CSV and PDF reports

## Tech Stack
- Backend: FastAPI + Python
- Computation: NumPy + SciPy
